# Maximum Clique Detection using MIPS Assembly

## Description
This project implements a maximum clique detection algorithm using MIPS Assembly language.

## Project Tasks
- Read graph from input text file
- Detect maximum clique using brute-force approach
- Write results to output text file

## Files
- main.asm → MIPS assembly code
- input.txt → graph input
- output.txt → program output
